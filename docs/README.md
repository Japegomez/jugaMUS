# Sitio legal (GitHub Pages)

Páginas estáticas para Google Play y enlaces públicos.

## Activar GitHub Pages

1. Repo en GitHub → **Settings** → **Pages**.
2. **Build and deployment** → Source: **Deploy from a branch**.
3. Branch: `main` (o `develop` si prefieres), folder: **`/docs`**.
4. Guardar. En 1–2 minutos la web estará en:
   `https://japegomez.github.io/jugaMUS/`

(Sustituye `jugaMUS` por el nombre real del repo si difiere.)
